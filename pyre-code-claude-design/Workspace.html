#09 Causal Self-Attention

Implement the causal self-attention operation — the masked version of scaled dot-product attention used in every decoder-only language model. Queries may only attend to keys at their own position and earlier.

Given q, k, v of shape (B, H, T, D), compute the attention output of the same shape. You must:

  1. Compute scaled dot-product scores.
  2. Mask future positions with -inf before softmax.
  3. Return softmax(scores) @ v.

Formula

Attention(Q,K,V) = softmax( Q·Kᵀ / √d + M ) · V
M[i,j] = 0 if j ≤ i else −∞

Examples

q.shape(1, 4, 8, 16)
k.shape(1, 4, 8, 16)
v.shape(1, 4, 8, 16)
out(1, 4, 8, 16) — row i sees only keys 0..i

Hint

⚑ Hint Use torch.triu(torch.ones(T, T), diagonal=1).bool() to build the causal mask once, then broadcast it across batch and head dims. Do not create a new mask every forward pass in production — but for this test, do.
